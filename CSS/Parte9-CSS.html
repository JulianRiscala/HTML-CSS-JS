CSS

Linkear al HTML

Para linkear un achivo .css, debemos hacerlo con la etiqueta link seguido de rel="stylesheet" type="text/css" href="ejemplo.css", siempre fijarse en que carpeta esta el archivo.

Estructura basica

Cuando empezamos, debemos colocar primero el selector, luego una propiedad y un valor para cada propiedad:

selector {
     propiedad: valor;
}

Selectores

Hay varios tipos de selectores:

Universal:
El universal se pone el signo * (asterisco), Abarca todos los elementos de la pagina web.
De tipo:
Seleccionamos por elementos. Buttons, h1, h2, etc.
Clases:
Para empezar debemos darle un atributo class a alguna etiqueta o elemento. class="ejemplo". Y en css el selector empieza con un punto, seguido de la clase.
Toda esta lista va a tener una clase.
ID:
Tal vez no queremos cambiar todos los elementos de tipo p para un mismo color, podemos darle un id para hacerlo en un parrafo especifico.
En la hoja .css la semantica para seleccionarlo seria: #id-ejemplo
El id debe ser unico!
Por atributo:
No hay muchas explicaciones pero aprendi que se pueden inventar atributos a los elementos recien ahora ¯\_(ツ)_/¯
En la hora css debemos poner el nombre del atributo entre [atributoinventado="salsa"]
Descendiente:
Tal vez tenemos un parrafo dentro de un h2 (por ejemplo) y solo queremos modificar el formato del parrafo, pero no el elemnto h2 que lo contiene.
A la vez no queremos modificar el formato de todos los parrafos!!! Esta es la mejor manera.
En CSS primero ponemos el contenedor, y luego el contenido que queremos cambiar. EJEMPLO: h2 p {...}
Pero estos elementos tambien pueden ser cambiados por atributos, id, etc.
En este caso, esta lista tiene una clase, pero solo cambie de color los terminos y no las definiciones.
Pseudo-clases:
Existen ya varias caracteristicas que podemos darle a los elementos, son algo asi como eventos.
Se introducen con: etiqueta:pseudo-clase. Un ejemplo puede llegar a ser hover, que cambia de color la etiqueta cuando pasamos el mouse encima.

Tenemos que tener en cuenta que podemos mezclar varios tipos de selectores para ser cada vez mas especifico.

Un ejemplo puede ser:

* {
     color:red;
}

Tambien podemos (en vez del color textual) poner el valor hexadecimal.

Metodologia BEM

Metodologia para evitar conflictos en jerarquia vamos a usar la metodologia BEM que se rige por clases. Hay mas metodologias pero antes que nada.
Lo que vamos a hacer, es darle a un elemento una clase class="noticias-form",
y dentro de cada bloque de elemento seguiremos la misma clase siguiende otro nombre. class="noticias-form__p" o class="noticias-form__input".
Vemos como usamos doble guion bajo para separar los sectores de los elementos. PERO SOLO POR UN SECTOR.
Y si por ejemplo tenemos varios inputs y uno en especifico queremos cambiar el color o etc. le pondremos doble guien medio con un nombre que queramos. class="noticias-form__input--nombre"

Unidades de medida

Existen 2 tipos de medidas, relativas (lo cual se portea bien a celulares) y fijas (Pixeles, mm, cm).

Tipografia

Con la propiedad font-size podemos elegir la medida que queramos para la fuente de un texto.

Medidas fijas:
  • px (pixeles)
  • cm
  • mm
  • pt (puntos)
Medidas relativas:
  • em
  • View Port
    • width que es el ancho: vw
    • height que es el alto: vh

em

Un em equivale a 16 px (por defecto). Darle 5em a una fuente es en total 80px.
Pero podemos elegir esa medida relativa por herencia. Y eso lo hacemos cambiando el valor de la font-size a la clase madre.
Pero tambien puede ser la medida de cualquier propiedad que queramos cambiar.
En la planilla CSS este ejemplo se puede ver en el articulo sobre las abejas.


View Port

Estas medidas ocupan todo el ancho y alto de la pantalla. Van del 0 al 100 que seria como porcentajes, pero no son porcentajes.

Propiedades de texto

Estas son las propiedades mas comunes:

  • font-size: Que basicamente ya esta explicada mas arriba
  • font-family: Esta es la tipografia
  • line-height: Esto es espacio vertical que ocupa en total, por defecto tiene 1, con 2 ocuparia el doble de espacio PERO NO ES MAS GRANDE.
  • font-weight: Este es el ancho que se estira la letra, se engorda.

Tambien podemos utilizar tipografias externas, podemos buscarlas en Google Fonts, exportamos el link href y la modificamos con font-family (la coma en la propiedad sirve para tener una letra back up).
