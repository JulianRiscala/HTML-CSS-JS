Normalice

Las distancias entre margenes de textos, los br, todo lo que nosotros no midifiquemos, viene por defecto por el navegador.
Esto es un problema cuando varios navegadores quieren abrir una pagina web que no esta preparada para funcionar en varios navegadores.
Entonces nosotros podemos "normalizar" nuestra pagina web y que una hoja de estilo CSS se encargue de todo eso.
Aca dejo el link para el archivo normalize.
Ahora lo que veo en este tutorial es que cambia un par de cosas en el normalize que las dejo aca escritas para futuros usos?


  • Ir al selector de tipo img y agregar max-width: 100%;. Para que las imagenes en celulares se ajuste.
  • Crear un selector universal * y agregar box-sizing: border-box;. Es por una cuestion de cajas que todavia no sé ¯\_(ツ)_/¯ pero es importante.
    En este mismo selector agregamos padding: 0; y margin: 0;. Es mejor poner 0 sin medidas porque trabaja como un multiplicador. X * 0 = 0.

Una vez hecho este proceso de normalize, ya podemos trabajar mas tranquilos y al 100% con CSS. YA ME SACARON LOS PUNTOS DE LOS ITEMS DE LAS LISTAS Y ME PONGO RE MAL LOCO.

CAJAS

Esto es de HTML pero la verdad es que no lo estudie... Se que hay etiquetas que ocupan todo el ancho de la pantalla, y todo el ancho de la pantalla seria la etiqueta body, donde metemos todo.
Es decir que una etiqueta h1, h2, p, etc ocupa todo el ancho de la pantalla, no se puede poner un h1 al lado de otro h1. Estos se llaman ETIQUETAS EN BLOQUE
Pero hay etiquetas que si se pueden poner al lado de la otra como la etiqueta b, i y otras que modifiquen el texto. Y ESTOS SE LLAMAN ELEMENTO EN LINEA (inline)

Ahora con CSS podemos hacer que estos elementos se comporten como nosotros queramos, en linea o en bloque.
A las etiquetas en bloque podemos ponerles una propiedad display para que se conviertan en linea, con el atributo inline. Ejemplo abajo.

Padding

El padding es la distancia entre el texto y su caja. Hay 4 paddings en total: izquierda, derecha, encima y por debajo. O left, right, top y bottom.
Es una propiedad que podemos acortar. Aplicaremos esto en un H1.

Hola

Tenemos varias maneras de aplicar esta propiedad :

  • padding: 10px; esto aplica el padding de 10 pixeles en los 4 sentidos.
  • padding: 10px 20px; esto aplica el padding de 10 pixeles en top y bottom, y despues 20 pixeles a left y right.
  • padding: 10px 20px 15px 30px; esto aplica el padding de 10 pixeles a top, 20 a right, 15 a bottom y 30 a left. Osea en sentido a las agujas del reloj. Podemos omitir el ultimo numero y se aplicara el numero del medio a right y left. Tambien podemos discriminar cada lado del padding con padding-bottom, padding-left, etc.

width y height

Bueno, usar estas propiedades junto a padding es medio raro... en el h1 de arriba tenemos padding: 10px 50px, height: 100. width: 100px.

  • SI TENEMOS EL box-sizing: border-box; que asi esta por default en el normalize, el color de fondo quedaria cuadrado segun las medidas del height y el width.
  • SI CAMBIAMOS A box-sizing: content-box; el color ya no es cuadrado con las medidas de height y width, si no que toma en cuenta el padding y lo suma en 4 lados.
    Seria en total 120 de height y 200 de width. Ya que suma 10 en top/bottom y 50 a los costados

Box model

Es el modelo de las cajas. Son los 4 atributos que forman el box model y tienen un orden de jerarquia

  • Content: que es el contenido y se puede modificar con la propiedad line-height
  • Padding
  • Border
  • Margin

Margin

El margen entre elementos se puede setear con la propiedad margin, clavo un ejemplo abajo. Ambos h1 tienen 10 de margin cada uno, pero la division entre ellos es un poco mayor porque se suman sus ambos margenes.
Podemos usar margin: auto para centrar la caja en el eje X.

Bordes

Con border podemos aplicar los estilos de bordes que queramos, con 3 atributos, la medida, el tipo de borde y un color.
Para redondear los bordes de los elementos podemos usar el atributo border-radius. Para que sea totalmente redondo tenemos que setearlo al 50%.

Box shadow

Podemos usar la propiedad box-shadow para agregarle sombra a la caja. Tiene 5 atributos:

  • Cuanto se mueve en eje X
  • Cuanto se mueve en eje Y
  • Tamaño del desenfoque
  • Cuanto borde (que generalmente es 0)
  • Color

Text shadow

Con la propiedad text-shadow podemos darle sobra al texto que esta dentro y consta de 4 atributos:

  • Cuanto se mueve en eje X
  • Cuanto se mueve en eje Y
  • Tamaño del desenfoque
  • Color
Ambas propiedades de shadow pueden intensificarse si en el codigo CSS agregamos una coma, se repite el codigo, coma y asi hasta finalizar con el punto y coma.

Rotacion

Podemos usar la propiedad transform: rotate(gradodeg) para rotar la caja.
