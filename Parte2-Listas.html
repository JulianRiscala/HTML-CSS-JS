Listas!

HTML nos proveé varios tipos de listas:

Listas ordenadas:

Cada item esta enumerado, util para una guia de pasos para una receta, o contratos legales donde cada punto debe estar enumerado, indices, etc.

Listas sin orden:

Cada item empieza con un "bullet point".

Listas con definiciones:

Son para sets de terminos y sus definiciones.

Listas ordenadas

Son creadas con la etiqueta ol (de ordered lists) y cada item empieza con la etiqueta li (de list item). UN EJEMPLO:

  1. Prender la computadora.
  2. Abrir Steam.
  3. Abrir Cookie Clicker.
  4. Replantear tu vida.
  5. Click a la galletita.
  6. Disfrutar la dopamina al ver numeros mas grandes.

Listas sin orden

Se crean con la etiqueta ul (de unordered lists) y cada item se abre con la etiqueta li. EJEMPLO:

Lista para un cigarro:

  • Paquete de tabaco.
  • Paquete de papelillos.
  • Paquete de filtros.
  • OPCIONAL: Bandeja para colocar el tabaco.

Listas con definiciones

Se crean con la etiqueta dl (de definition lists) y cada item se abre con dt (de definition term) seguido de la etiqueta dd (de definition). EJEMPLO:

Partes de un PC:

Procesador:
Es el hardware dentro de un computador, teléfonos inteligentes y otros dispositivos programables.
Placa madre (Motherboard):
Es una tarjeta de circuito impreso a la que se conectan los componentes que constituyen la computadora.
Placa de video:
dispositivo de hardware que, en una computadora, interviene en el procesamiento de los datos que proceden de la CPU y los convierte en información que puede representarse a través de un dispositivo de salida.
Memoria RAM:
La RAM se cargan todas las instrucciones que ejecuta la unidad central de procesamiento (CPU) y otras unidades del computador, además de contener los datos que manipulan los distintos programas.
Fuente de alimentacion:
La fuente de alimentación o fuente de potencia es el dispositivo que convierte la corriente alterna (CA), en una o varias corrientes continuas (CC), que alimentan los distintos circuitos del aparato electrónico al que se conecta

Listas anidadas

Se pueden anidar listas de la etiqueta: li. Pero debe estar dentro de la misma etiqueta.

Lista para un cigarro:

  • Paquete de tabaco.
    • Tabaco mentolado
    • Tabaco sin aditivos
    • Tabaco saborizado
  • Paquete de papelillos.
  • Paquete de filtros.
  • OPCIONAL: Bandeja para colocar el tabaco.
