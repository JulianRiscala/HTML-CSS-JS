Tablas

Organizacion

La etiqueta table crea una tabla.
Se habre una nueva fila con tr, y cada dato se ingresa con td (data table) que refiere a cada columna.

Aunque una celda no contenga informacion, se debe abrir y cerrar la etiqueta td para que se renderize bien la tabla.

15 45
45 99999999 135
80 1 0

Los datos mas grandes van a acomodar todos los demas para que quede todo alineado.

Titulos para columnas y filas

Con la etiqueta th podemos poner titulos, es lo mismo que td pero ayuda/guia para identificar en el codigo los titulos de filas y columnas.

Se puede agregar una propiedad scope para identificarlos mas facil.

Ocupar varias celdas

Luego de abrir la etiqueta td, podemos usar la propiedad colspan="cantidad" para ocupar varios espacios de columnas.
O la propiedad rowspan="numero" para hacer lo mismo pero con filas.

Lunes Martes Miercoles Jueves Viernes
8:00 Matematicas Historia Filosofia Arte Matematicas
9:00 Arte Naturales Sociales Portugues
10:00 Recreo Matematicas
11:00 Hora libre

En el ejemplo "recreo" ocupa 4 columnas y 2 filas, pero como esta alineado a la izquierda, y no hay sombras, no se ve tanto.

Tablas laaaaaargas

Hay 3 elementos que pueden ayudar a distinguir el contenido de las tablas. A parte que facilitan el uso de CSS.

thead

Los titulos superiores de las tablas se meten en el elemento thead.

tbody

El cuerpo entero de la tabla debe estar dentro del tbody.

tfoot

"Pie de pagina" pertenece al tfoot

El tfoot puede usarse como una suma total de los elementos del body por ejemplo.
