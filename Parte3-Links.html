Este diagrama es una ayuda para organizar el proyecto, la carpeta ElClan es nuestra root folder. Todo lo que este dentro de esta carpeta contendra archivos para la pagina web.

Relaciones

Las relaciones dentro de los archivos y carpetas siguen las terminologias como arboles genealogicos. La carpeta ElClan es padre de Inicio.html y carpetas como imagenes,productos,etc.

Paginas principales

Las paginas principales de un sitio html (y las paginas principales de cada seccion) son llamadas Inicio.html o Index.html.
Los servicios web usualmente vuelven a la pagina principal si no esta especificado el link.
Entonces si ponemos ElClan.com nos llevara a ElClan.com/Inicio.html.html.
Y ElClan.com/Productos nos llevara hacia ElClan.com/Productos/Inicio.html.

URLs relativos

Utilizados para moverse entre links de tu propio dominio, son mas cortos que los URLs absolutos y tienen propieades que se deben respetar para poder moverse entre carpetas.

  • Misma carpeta: Solo se utiliza el nombre del archivo
    • a href="ejemplo.html">Ejemplo
  • Carpeta hija: Se ingresa la carpeta hija -> / -> nombre del archivo
    • a href="Mesadas/Inicio.html">Inicio
  • Carpeta nieta: Se ingresa carpeta nieta -> / -> carpeta hija -> / nombre del archivo
    • a href="Productos/Muebles/Inicio.html>Inicio
  • Carpeta madre: Se utiliza ../ para indicar que es una carpeta por encima.
    • a href="../Inicio.html>Inicio
  • Carpeta abuela: Se repite ../ para indicar que son dos carpetas por encima.
    • a href="../../Inicio.html>Inicio

El metodo ../ se puede repetir las veces necesarias para subir de carpeta.

Links hacia mails

Luego de abrir la etiqueta podemos colocar href="mailto:ejemplo@mail.com">label. Podremos mandar un mail al mail ejemplo.

Abrir un link en una pestaña nueva

Luego de colocar la referencia en la etiqueta, podemos poner target="_blank", cerrar la etiqueta y el nuevo link se abrira en una nueva pestaña

Links a partes especificas de la misma pagina web

Es un recurso util para viajar entre las distintas secciones de la pagina web.
Antes que nada debemos darle un id a la etiqueta que queramos linkear. Puede ser cualquier etiqueta HTML
Se recomienda que las id empiecen con un guion bajo o una letra, NO un numero.
Luego en la etiqueta del link, en el href se ingresa entre comillas el # + ID

Inicio de la pagina

Tambien se puede linkear a partes especificas de OTRA pagina!
Para hacer eso la href con la pagina web tiene que seguir asi: .com/#inicioP1
