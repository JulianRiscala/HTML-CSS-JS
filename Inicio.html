Superíndice y subíndice

Las etiquetas sup (superíndice) y sub (subíndice) se pueden utilizar... para...
Bueno la etiqueta sup para mostrar numeros con potencias, tipo 22 = 4. Ojo.
Y la etiqueta sub ni idea, asi que voy a usar el ejemplo del manual: The amount of CO2 in the atmosphere grew by 2ppm in 20091.

Salto entre lineas

Bueno, sabemos que para saltos entre lineas se debe usar la etiqueta br.
Esta etiqueta pertenece al tipo de etiquetas vacias. Pero para dar un salto entre lineas con la intencion de cambiar de tema debemos usar la etiqueta hr (lo ideal es cerrar el parrafo primero).

Margen semántico

Strong

A veces debemos darle enfasis (con la etiquetastrong) al texto que estamos escribiendo, la etiqueta strong indica la importancia del texto dandole un formato en negrita

Emphasis

La etiqueta em realiza un cambio muy sutil a la oracion. Como podras notar, hace el mismo efecto tipo cursiva.

Abreviaciones y acronismos

La etiqueta abbr se utiliza con un titulo que sea la palabra completa. EJEMPLO: Prof. Julian.

Citas y definiciones

Podemos usar la etiqueta cite para referirnos a citas sobre paginas webs o libros tal como:

La Guia de la Vida por Bart Simpson a salvado millones de personas de la depresion gracias a sus trucazos.

Tambien existe la etiqueta dfn para explicar una nueva terminologia:

La yerba mate es una especie Árbol neotropical originaria de las cuencas del Alto Río Paraná, Alto Río Uruguay y algunos afluentes del Río Paraguay​.

Detalles del autor

La etiqueta address tiene un uso especifico: contener los detalles de contanto del autor sobre una obra, tambien numeros de telefonos o direcciones de mail.
El uso de la etiqueta address es la siguiente: ->Address ->p ->a href="mailto: direccion de mail o pagina web. -> cierro a -> cierro p y sigo en otro ->p.

julianriscalatrullas97@hotmail.com

Tigre, Buenos Aires.

Cambios de contenido

La etiqueta del tacha el contenido que este dentro y la etiqueta ins lo subraya. Estas etiquetas se usan idealmente cuando una parte del documento fue eliminado para ingresar otra cosa:

La epoca nazi fue lo mejor peor que se vivio en el siglo XX.

Y la etiqueta S se utiliza cuando se quiere mostrar algo ya no es relevante actualmente. Tacha el contenido igual que del pero es otro uso:

Entrada al cine: $300 actualmente $250.
