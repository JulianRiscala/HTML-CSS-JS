Semantica correcta para HTML

Header

El header es como la barra de navegacion de todo el dominio. Se puede utilizar para copiar y pegar en varias paginas web.
Por ejemplo una barra de busqueda, o lista de links para navegacion (para esta parte podemos usar la etiqueta nav).
En este ejemplo, hago uno y lo pego en todas las otras paginas.

Article

Podemos utilizar la etiqueta article para crear articulos en nuestra pagina web, una manera mas de organizar toda la semantica.
Dentro podemos usar la etiqueta section para crear secciones mas ordenadas.
Al final podemos usar un aside (fuera del article) que es un lado B del articulo. Con CSS podemos hacer que apareza a un costado, solo HTML lo tira para abajo.
El footer se utiliza para el pie de pagina, pueden ser los ultimos links de las paginas web con cosas legales, contactos, etc.
