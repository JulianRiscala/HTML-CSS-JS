Display

Para este momento ya deberiamos tener en cuenta que hay varios tipos de elementos con display: block, inline y la mezcla inline-block.
Que los blocks son en bloque, ocupan todo el ancho.
Que inline son en linea, que ocupan solo lo que tengan en el contenido.
Y despues inline-block son la mezcla, podemos darle tamaños, posiciones e inclusive pueden no tener nada dentro.

Overflow

Con el overflow podemos crear cuadros con texto dentro, y que si el texto excede del tamaño del cuadro o caja, tenga un scroll incluido para bajar por el contenido.

  • De base esta en inherit
  • Podemos colocar auto para que podamos scrolear por la caja
  • Y scroll si a si pone la barra para scrolear, a diferencia de auto que primero detecta la necesidad de una.
  • overflow-y:auto seria lo mismo que overflow auto.

Un uso que podemos darle a la propiedad float es darle una imagen articulo a un div.
Para hacer este efecto, primero va la img dentro del div, y luego el texto pero suelto, sin estar dentro de un parrafo ni nada!

Pseudo-Elementos

Tenemos 2 pseudo-elementos que podemos aplicar en cualquier tipo de elemenos MENOS INLINE. y deben seguir del elemento en css con :: .

  • - ::first-letter aplica propiedades a la primera letra del texto.
  • - ::first-line hace lo mismo pero a la primera linea del texto, nomas si se hace mas chica la pantalla, la primera linea puede variar.
  • - ::placeholder funciona para cuando los inputs tienen un texto de fondo antes de escribir algo, con esta pseudo-propiedad cambiamos el formato de ese texto.
  • - ::selection aplica propiedades cuando seleccionamos partes del texto.
  • - ::after es de elementos de hijos, si o si necesita un padre porque no forma parte del dom.
  • - ::before funciona de la misma manera, ambos pseudo-elementos deben contener un content.

Pseudo-Clases

  • - :lang(abreviacion de un lenguaje) aplica las propiedades siempre y cuando en el html hayamos definido con lang="abreviacion de un lenguaje". Casi nunca se usa.

Object fix

La propiedad object-fit define a un elemento (foto o video) por como debe responder al height y width de la caja contenedora. Nos permite controlar como queremos que la imagen se ajuste a la caja.
Y tiene varios valores:

Aca una imagen de modelo 350px X 350px:
  • - fill es el value default, estira la imagen para llenar la caja contendora. Podemos apreciar como la imagen se distorciona para llenar a la caja.
  • - contain incrementa o disminuye el tamaño de la imagen para llenar la caja preservando su relacion de aspecto (aspect-ratio) original
  • - cover la imagen tapa toda la caja contenedora pero al mantener su aspect-ratio, puede verse recortada la imagen. La diferencia con contain, es que contain cubre ancho o alto de la caja, pero puede sobrar espacio en blanco sobre la caja.
    En este ejemplo tenemos la propiedad object-position:right; para mostrar el lado derecho de la imagen.
  • - none hará que la imagen ignore las medidas de la caja y se muestre en su resolucion original.
  • - scale-down lo que hace es comparar entre none y contain, y se queda con la resolucion mas chica.
